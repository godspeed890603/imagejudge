## 目的說明
在2018年，因應公司規劃AOI自動判CODE評估立達軟體科技R8影像辨識系統，為使自己可以詳細深入了解，並擁有可以更詳盡評估的能力，自行學習簡易判CODE程式。</br></br>

3.因應工廠提供的資料不足，採用defect特徵切割的方式，增加sample量，讓labeling可以有更多的資料</br></br>
